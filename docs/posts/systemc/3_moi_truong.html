        # Thiết lập môi trường trên vscode

        Môi trường chạy SystemC được hổ trợ trên hầu hết các EDA phổ biến như ModelSim, Vcs, QuestaSim. Về mặt kiến trúc như đã đề cập ở phần Giới thiệu, căn bản chúng ta có thể chạy SystemC trên bất kỳ Development Tool nào hổ trợ C++. Mình xin chia sẽ môi trường trên WSL và Vscode như sau:

        # Yêu cầu:

        • Hệ điều hành: win_10 hoặc win_11
        • Cài đặt WSL (Tham khảo: https://windowsloop.com/enable-wsl-windows-10-home/ (opens new window))
        • Cài đặt Vscode (Tham khảo: https://code.visualstudio.com/ (opens new window) )
        • Tải về SystemC library (Tham khảo: https://www.accellera.org/images/downloads/standards/systemc/systemc-2.3.3.zip (opens new window))

        # Compile và cài đặt SystemC library trên WSL:

        1. Cài đặt C++ compiler và Gdb cho WSL. Tham khảo: Bước 4,5,6 https://code.visualstudio.com/docs/cpp/config-wsl (opens new window)
        • Tại wsl terminal, cài đặt compiler:
            sudo apt-get update
    sudo apt-get install build-essential gdb
    whereis g++
    whereis gdb
        1. Giải nén SystemC library. Truy cập đến SystemC library bằng wsl. Tham khảo cài đặt tại file INSTALL hoặc có thể thực hiện các bước sau:
        2. Tạo thư mục objdir: mkdir objdir
        3. Di chuyển đến objdir: cd objdir
        4. Chọn g++ compiter: export CXX=g++
        5. Tiến hành thiết lập cài đặt bằng lệnh: ../configure --prefix=/usr/local/systemc-2.3.3 --enable-debug
        6. Tiến hành compile bằng lệnh: make
        7. Cài đặt : make install

        # Chạy example trên vscode:

        1. Cài đặt C/C++ extension https://marketplace.visualstudio.com/items?itemName=ms-vscode.cpptools (opens new window)
        2. Tải hello example code tại: https://github.com/HaTiDe/SystemC_Example (opens new window)
        3. Mở vscode -> File -> Open Folder -> chọn hello . Tiến hành kết nối vscode với WSL
        4. Compile và chạy chương trình mẫu:

        kết quả: Hello World
